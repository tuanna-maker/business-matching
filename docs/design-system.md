## Icons
Using Lucide Icons (https://lucide.dev)
- Menu, Search, Bell, User, Settings, Dashboard, Projects, Matches, Messages, Analytics
- Arrow icons, Action icons (Edit, Delete, Save), Status icons (Check, X, Alert)

## Components

### Buttons
- Primary Button: Blue background, white text, rounded corners
- Secondary Button: White background, blue border, blue text
- Ghost Button: Transparent background, blue text
- Icon Button: Square button with icon only

### Cards
- Elevated Card: White background, subtle shadow, rounded corners
- Outlined Card: White background, gray border, rounded corners

### Inputs
- Text Input: Rounded corners, border focus ring
- Textarea: Same as text input but taller
- Select: Dropdown with search capability
- Checkbox/Radio: Custom styled with blue accent

### Navigation
- Sidebar: Collapsible on mobile, icons with labels
- Top Bar: Logo, search, notifications, user menu
- Breadcrumbs: Horizontal navigation trail

### Feedback
- Toast: Slide-in notifications at top-right
- Modal: Centered overlay with backdrop
- Tooltip: Small informational popup on hover

## Animation Principles
- Duration: 150ms for micro-interactions, 300ms for page transitions
- Easing: ease-in-out for smooth motion
- Stagger: 50ms delay between sequential animations
